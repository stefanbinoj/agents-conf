# Camdiv :  
I am building Camdiv, a free random video chat platform that connects users with strangers worldwide for real-time video conversations. The platform offers multiple ways to connect:
- Video Chat - Face-to-face video calls with random strangers
- Webcam Chat - Direct webcam-based communication
- Voice Chat - Audio-only conversations
- Text Chat - Real-time messaging
The app features real-time matchmaking to pair users, voice messages, screen sharing, emoji support, and a modern chat interface. Users can join rooms and be matched with random partners instantly for conversations.
